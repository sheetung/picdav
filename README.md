# 图片上传工具

基于 WebDAV 的图片上传与管理工具。

## 架构

两个独立服务，共用同一份配置：

| 服务 | 端口 | 定位 | 接口 |
|------|------|------|------|
| **展示页** (`gallery.py`) | **5000** | 纯浏览，适合公网暴露 | `/` `/api/images` `/api/proxy` |
| **管理端** (`upload.py`) | **5001** | 完整功能，建议内网使用 | `/` 上传/配置/图片库/删除 |

## 快速开始

```bash
pip install -r requirements.txt

# 终端 1：展示页 (http://localhost:5000)
python gallery.py

# 终端 2：管理端 (http://localhost:5001)
python upload.py
```

## 配置

在管理端（5001）页面中填写 WebDAV 信息，配置保存在 `config/image_uploader.json`，密码使用 Fernet 加密存储。

## 功能对比

| 功能 | 展示页 (5000) | 管理端 (5001) |
|------|:-:|:-:|
| 浏览图片 | ✓ | ✓ |
| 上传图片 | - | ✓ |
| 配置 WebDAV | - | ✓ |
| 删除图片 | - | ✓ |
| 图片代理 | ✓（限绑定域名） | ✓ |
| 拖拽/粘贴上传 | - | ✓ |

## 公网部署说明

`gallery.py`（5000）可暴露到公网：

- **无敏感接口** — 没有配置、删除、上传端点
- **代理受限** — `/api/proxy` 只允许代理已配置的 WebDAV 服务器上的图片
- **无认证信息泄露** — 不暴露任何配置数据

建议配合 Nginx/Caddy 反向代理加 HTTPS。
